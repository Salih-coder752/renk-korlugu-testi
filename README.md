# Color vision Test App
A simple and interactive web application designed to help users test different types of color vision deficiencies such as Deuteranopia, Protanopia, and Tritanopia. Built with accessibility and user experience in mind.

 Features
 Multiple color blindness test modes (Ishihara plates, color matching, etc.)
 Real-time test result preview
 Responsive and accessible design for all devices
 Educational insights about different types of color blindness
 Tech Stack
Frontend: HTML5, CSS3, JavaScript (or React, if applicable)
Styling: Custom CSS / Tailwind / Bootstrap (adjust based on your project)

 How It Works
Users are shown a series of color-based patterns or images.
They’re asked to choose what number or shape they see.
The app then analyzes answers to estimate the type and degree of color vision deficiency.m startHosting: GitHub Pages / Netlify / Vercel Lightweight front-end with no external dependencies (optional customization)
